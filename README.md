# Student Performance Early Warning System

An ML-powered system to identify students at risk of academic failure 
before final exams, enabling early intervention by lecturers and advisors.

## Status
🚧 In development — Day 1: project setup complete.

## Planned Features
- Risk prediction using ML (Logistic Regression, Random Forest, Neural Network)
- Explainable predictions (feature importance / SHAP)
- Interactive Streamlit dashboard
- Actionable intervention recommendations
